# BodyCoreProduct

### Предназначение
Данный сервис позволит вам:
1) определить процент жировой ткани в организме;
2) составить комфортный для организма план снижения веса.

Существует 6 типов телосожения:
- Истощение
- Атлетическое телосложение
- Хорошая физическая форма
- Средний уровень жира
- Наличие лишнего веса
- Ожирение

И ключевым показателем является именно процент жировой ткани, а не вес.
Так, например, у спортсменов может быть большая масса тела, но невысокий % жира в организме. 
Здесь вы сможете рассчитать свой показатель.

### Форма ввода показателей.
Простая форма для среднестатистического человека + 3 дополнительных поля (объемы шеи, талии и бедер) для людей, активно занимающихся спортом.
![Рис. 1](https://github.com/DoNatPanic/BodyCoreProduct/raw/main/BodyCoreProduct/Visualize/inputform.png)

### Табличные результаты составленного плана снижения веса
Нулевая неделя - ваша норма КБЖУ для поддержания веса в текущем состоянии
Все последующие недели с новой нормой КБЖУ для "безопасного" снижения веса, который не приведет к нарушению обмена веществ и срывам.
![Рис. 2](https://github.com/DoNatPanic/BodyCoreProduct/raw/main/BodyCoreProduct/Visualize/tableresults.png)

### Визуализация результатов
Различными цветами обозначаются границы нахождения в той или иной зоне телосложения.
![Рис. 3](https://github.com/DoNatPanic/BodyCoreProduct/raw/main/BodyCoreProduct/Visualize/graphicsresult.png)


### Дисклеймер: 
Данный сайт не направлен на популяризацию худобы и навязывание стандартов. 
Есть мнение что, у каждого человека есть такой вес, в котором ему максимально комфортно вести активную жизнь.
И если по какой-то причине его вес выше данной отметки, возможно, данный сайт сможет стать инструментом к ее достижению.

Также, перед применением рекомендаций желательно проонсультироваться с врачом.
